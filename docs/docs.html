contains(dest, src, dest_in_dict=False, case_sensitive=False)

参数

名称 说明 类型 默认值
dest 目标文件 str | list -
src 源文件 str -
dest_in_dict 目标文件是否在 dict 目录 bool False
case_sensitive 是否区分大小写 bool False

nlptools.contains("a.txt", "b.txt")
